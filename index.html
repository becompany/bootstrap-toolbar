Bootstrap Toolbar

Enhanced toolbar for Twitter Bootstrap.

Options

Add data-toolbar-toggle="button" on a button to enable checkbox style toggling.

Events

select.bs.toolbar
Fired on the selected button. The selected element is passed as second parameter.
unselect.bs.toolbar
Fired on the previously button. This event is triggered on toggle enabled buttons only. The unselected element is passed as second parameter.
